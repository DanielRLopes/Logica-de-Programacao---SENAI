# 💡 Banco de Ideias

Espaço para capturar **ideias**, **referências** e **snippets** de código.

---

## 📂 Estrutura

| Pasta/Arquivo | Propósito |
|---------------|-----------|
| `backlog.md` | Lista mestra de todas as ideias |
| `snippets/` | Trechos de código reutilizáveis |
| `referencias/` | Links, tutoriais, artigos úteis |

---

## 📝 Backlog de Ideias

O arquivo [`backlog.md`](./backlog.md) contém todas as ideias capturadas, organizadas por prioridade e status.

### Categorias de Ideias

| Categoria | Descrição |
|-----------|-----------|
| 🚀 Alta | Implementar em breve (próximas 2 semanas) |
| 📌 Média | Implementar quando possível (próximos 2 meses) |
| 💭 Baixa | Algum dia, talvez |
| ✅ Concluída | Ideia implementada (mover para `PROJETOS/`) |

---

## 🔧 Snippets

Trechos de código prontos para reutilizar em futuros projetos.

### Exemplos de Snippets
- `auth-jwt.js` — Configuração de autenticação JWT
- `conexao-db.js` — Conexão com banco de dados
- `validacoes.js` — Funções utilitárias de validação
- `middlewares.js` — Middlewares comuns

### Como Adicionar
1. Crie arquivo em `snippets/` com nome descritivo
2. Adicione código bem comentado
3. Inclua exemplo de uso no topo do arquivo

---

## 📚 Referências

Links úteis organizados por tema.

### Categorias Sugeridas
- Tutoriais
- Documentações
- Artigos técnicos
- Vídeos/Aulas
- Ferramentas úteis

### Como Organizar
Crie um Markdown por tema em `referencias/`:
- `frontend.md`
- `backend.md`
- `devops.md`
- `ia-ml.md`

---

## 💡 Dicas

- **Capture rápido:** Não filtre ideias na captura — filtre depois
- **Revise semanalmente:** Atualize status e prioridades
- **Conecte com projetos:** Ideias aprovadas viram projetos em `PROJETOS/`
